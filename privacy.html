Web browsers

The only web browsers I would use are GNU Icecat and ungoogled-chromium for firefox and google chrome alternatives respectively

Operating systems

Personally, the only operating systems I would use for day to day use would be any FSF endorsed distrobution, but realistically, any GNU/Linux distrobtion or *BSD varient should be okay. For mobile devices use an Android with either LineageOS or GrapheneOS Avoid Windows, MacOS, IOS and OEM Android versions at all costs.
