Setup

Homeserver

All my services I host run on a 2011 Mac mini server that I have situated on my desk. Alongside the occasional Thinkpad that I hookup to my "loop" These all run on Trisquel GNU/Linux

The webserver I use is Apache2, this is mainly due to me not being able to get nginx working...

Desktop

My PC is a self-built desktop that contains an RTX 3050, Intel i5 13500, and 32GB of DDR4 RAM. It currently runs Gentoo GNU/Linux on it.

Software

On my personal systems, for my window manager I run dwm with st and dmenu (I occasionally switch to rofi every now and then) my configurations for such can be found on my github. For my browser I use GNU Icecat with a few extra extensions. I use VSCodium and vim as code editors.
