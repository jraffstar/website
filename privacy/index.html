Privacy Software Alternatives

Web browsers

The only web browsers I would use are GNU Icecat and ungoogled-chromium for firefox and google chrome alternatives respectively. Alternatively, if you like vim keybinds Qutebrowser is also a solid choice.

Operating systems

Personally, the only operating systems I would use for any main personal day-to-day setup would be any FSF endorsed distribution, but realistically, any GNU/Linux distrobtion or *BSD varient should be okay. For mobile devices use an Android with either LineageOS or GrapheneOS. Avoid Windows, MacOS, IOS and OEM Android versions at all costs.

VPNs

Preferably self host, but if that is not in your interest, IVPN, OVPN or Mullvad are all good options.

Instant Messaging Services

Use Matrix, if you want a modern client with multimedia, screenshare, VC, and encryption, use the Element Client, if you don't like electron use nheko

Git hosting

Preferably self host using cgit, but if that is not an option, use notabug

Good Software Practices

If possible, avoid using software binaries and try to self compile everything. Not only does this increqse security, it can actually improve software's performance
